## 🏛️ Project Overview

This project transforms a static dataset of over 270,000 Olympic records (1896–2016) into a **living timeline**, exploring the intersection of biology, geopolitics, and sports history. The website is designed as a linear scrollytelling experience, guiding users through:

- 🌍 **Geopolitical Scrollytelling** — How historical events shaped global participation
- 🗺️ **Interactive Medal Map** — Explore Olympic performance across countries and time
- 🏃 **Morphological Specialization** — How athletes' bodies evolved by discipline
- ⚰️ **The Graveyard of Olympic Sports** — Discontinued and bizarre sports through history
- 🎯 **Find Your Sport** — An interactive tool matching biometric data to Olympic disciplines
